Sistem Informasi Desa (Si Desa) adalah aplikasi berbasis web yang dibangun dengan **Laravel** untuk mempermudah administrasi desa, mulai dari pengelolaan data penduduk hingga sistem pengaduan warga secara digital.

## âœ¨ Fitur Unggulan
- **Verifikasi NIK Otomatis**: Sistem akan mencocokkan NIK pendaftar dengan database master penduduk desa secara otomatis saat admin melakukan persetujuan akun.
- **Manajemen Wilayah**: Pengaturan data RT dan RW yang dinamis.
- **Dashboard Statistik**: Visualisasi data penduduk dan status pengaduan menggunakan Chart.js.
- **Sistem Pengaduan**: Warga dapat mengirim laporan/aspirasi dan memantau prosesnya (New -> Processing -> Completed).
- **Manajemen Akun**: Admin memiliki kendali penuh untuk menyetujui, menolak, atau menonaktifkan akun warga.
